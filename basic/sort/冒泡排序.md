# 冒泡排序

## 时间复杂度：最坏情况O(n*n)，最好情况O(N)

## 平均O(n*n)

## 下标从0开始

```c++
void BubbleSort1(int *r,int n){//冒泡排序
    int temp;
    for(int i=0;i<n;i++)
        for(int j=n-2;j>=i;j--){//从倒数第二个往前判断，效率更高
            if(r[j]>r[j+1]){
                temp=r[j];
                r[j]=r[j+1];
                r[j+1]=temp;
            }
        }
}
```
