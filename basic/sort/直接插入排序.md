# 直接插入排序

## 时间复杂度

### 最坏和平均O(n*n)

### 最优为O(n),在数列几乎有序时效率很高

## 每次把当前的数插入到一个有序的数组

```c++
void InsertionSort(int *r,int n){//直接插入排序
    int key,temp,j;
    for(int i=1;i<n;i++){//从第2项开始比,第一个单独就是有序的
        key=r[i];//当前要排序的数
        j=i-1;//从前一个位置开始比较
        while(j>=0&&r[j]>key){//将大于key的元素向后移,找到比key小的位置插入
            r[j+1]=r[j];
            j--;
        }
        r[j+1]=key;//插入到比key小的位置的后面
    }
}
```
