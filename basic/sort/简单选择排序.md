# 选择排序

## 时间复杂度

### 最坏、最优、平均都是O(n*n)，不稳定

## 每次找到第i小的元素（i到n中最小的元素），将这个元素与数组第i个位置上的元素交换

```c++
void SelectionSort(int *r,int n){//简单选择排序
    int minindex,temp;
    for(int i=0;i<n-1;i++){
        minindex=i;
        for(int j=i+1;j<n;j++){
            if(r[j]<r[minindex])
                minindex=j;
        }
        if(minindex!=i){
        temp=r[i];
        r[i]=r[minindex];
        r[minindex]=temp;
        //swap(r[i],r[minindex]);
        }
    }
}
```
