# 希尔排序

## 时间复杂度

### 最优O(n)

![](C:\Users\黄书恒1\Pictures\Code\屏幕截图 2024-06-05 110723.png)

```c++
void ShellSort(int *r,int n){//希尔排序O(nlogn)
    int j,gap,temp;
    gap=n/2;
    do{
        for(int i=gap;i<n;i++){
            temp=r[i];
            j=i;
            while(j>=gap&&r[j-gap]>temp){
                r[j]=r[j-gap];
                j-=gap;
            }
            r[j]=temp;
        }
        gap/=2;
    }while(gap>0);
}
```
