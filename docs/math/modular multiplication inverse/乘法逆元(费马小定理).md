# 乘法逆元(费马小定理)

### 若 p 为素数，gcd（a，p）=1，则$a^{p-1}\equiv1(\mod p)$。

#### 费马小定理 (要求模数必须是质数)而扩展欧几里得只要求 gcd（a，b）=1（互质）即可

```c++
#include<iostream>
using namespace std;
#define ll long long
ll a,ans,p,n;
ll ksm(ll x,ll y){//快速幂
    ll res=1;
    x%=p;
    while(y){
        if(y&1)
            res=res*x%p;
        x=x*x%p;
        y=y>>1;
    }
    return res%p;
}
int main(){
    cin>>n>>p;
    for(int i=1;i<=n;i++)
        printf("%lld\n",ksm(i,p-2));
    return 0;
}
//费马小定理 a,p互质那么a的p-1次方模p等于1,用这个式子替换ax模p等于1当中的1
//相当于求解a的p-2次方的值模p的值
```
