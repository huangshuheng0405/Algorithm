# bitset

#### 头文件

```c++
#include <bitset>
```

#### 指定大小

```c++
std::bitset<1000> bs;  // 1000位 全为0
```

#### 成员函数

| 方法              | 功能                                                         |
| ----------------- | ------------------------------------------------------------ |
| `b.any()`         | b中是否存在置为1的二进制位                                   |
| `b.none`          | b中不存在置为1的二进制数吗                                   |
| `b.count`         | b中置为1的二进制的个数                                       |
| `b.size()`        | b中二进制位的个数                                            |
| `b[pos]`          | 访问b中pos处的二进制位                                       |
| `b.test[pos]`     | b中pos位的二进制位是否为1                                    |
| `b.set()`         | 把b中所有二进制位都置为1                                     |
| `b.set(pos)`      | 把b中pos处的二进制位置为1                                    |
| `b.reset()`       | 把b中所有二进制位都置为0                                     |
| `b.reset(pos)`    | 把b中pos处的二进制位置为0                                    |
| `b.filp()`        | 把b中所有二进制位逐位取反                                    |
| `b.filp()`        | 把b中pos处的二进制取反                                       |
| `b.to_ulong`      |                                                              |
| `os<<b`           | 把b中的位集输出到os流                                        |
| `all()`           | 若所有位都是 `true` 则返回 `true`，否则返回 `false`          |
| `_Find_first()`:  | 返回 `bitset` 第一个 `true` 的下标，若没有 `true` 则返回 `bitset` 的大小**（下标从1开始）** |
| `_Find_next(pos)` | 返回 `pos` 后面（下标严格大于 `pos` 的位置）第一个 `true` 的下标，若 `pos` 后面没有 `true` 则返回 `bitset` 的大小 |

#### 运算符

`[]`访问特定的一位

注意：**`bitset` 只能与 `bitset` 进行位运算**，若要和整型进行位运算，要先将整型转换为 `bitset`。
